# CDN (Content Delivery Network)

Để bắt đầu, bạn cần thêm plugin TryOnAI vào ứng dụng web của mình. Plugin TryOnAI hoạt động cả trong môi trường CommonJS
và AMD (Asynchronous Module Definition) và có thể truy cập thông qua không gian tên global của TryOnAI khi được tải qua
một CDN (Content Delivery Network).

Khi tích hợp Plugin TryOnAI qua CDN, nên sử dụng phiên bản được tối ưu hóa cho sản xuất (minified). Bạn có thể chỉ định phiên bản bằng cách thêm nó vào đường dẫn CDN như sau:

- [Đối với một phiên bản cụ thể](https://tryonai.com/tryonai-plugin-1.2.0.min.js)
- [Đối với phiên bản mới nhất (tự động cập nhật)](https://tryonai.com/tryonai-latest.min.js)

# Tải Plugin

Để tải Plugin thông qua CDN, hãy thêm đoạn mã sau vào giữa các thẻ ```<head>``` của ứng dụng web của bạn:

```html
<!-- Đối với một phiên bản cụ thể: -->
<script src='https://yourURL/tryonhub.min.js'></script>

<!-- Đối với phiên bản mới nhất (tự động cập nhật): -->
<script src='https://yourURL/tryonhub.min.js'></script>
```

# Khởi Tạo file JS config
tạo file tryonhub.min.js ở trong thư mục public
```js
// This code is wrapped in an immediately invoked function expression (IIFE) to create a separate scope
!(function () {
    // Get all elements with the class 'try-on-hub-image-src'
    const elements = document.getElementsByClassName('try-on-hub-image-src');

    for (let i = 0; i < elements.length; i++) {
        // Create a new button element
        const button = document.createElement('button');
        button.classList.add('try-on-hub-btn');
        button.textContent = 'Try it on!';
        elements[i].appendChild(button);

        button.addEventListener('click', function (event) {
            // Find the closest ancestor element with the class 'try-on-hub-image-src' and get all the 'img' children
            const images = event.target.closest('.try-on-hub-image-src').getElementsByTagName('img');

            if (images.length > 0) {
                // Create a new div element for the overlay
                const overlay = document.createElement('div');
                overlay.classList.add('try-on-hub-ifrm-overlay');

                // Create a new div element for the container
                const container = document.createElement('div');
                container.classList.add('try-on-hub-ifrm-container');

                // Create a new div element for the header
                const header = document.createElement('div');
                header.classList.add('try-on-hub-ifrm-header');

                // Create a new paragraph element for 'Powered by'
                const poweredBy = document.createElement('p');
                poweredBy.classList.add('try-on-hub-ifrm-header-power-by');
                poweredBy.textContent = 'Powered by';
                header.appendChild(poweredBy);

                // Create a new paragraph element for 'TryOnHub.AI'
                const brand = document.createElement('p');
                brand.classList.add('try-on-hub-ifrm-header-brand');
                brand.textContent = 'TryOnHub.AI';
                header.appendChild(brand);

                // Create a new image element for the logo
                const logo = document.createElement('img');
                logo.src = 'https://www.tryonhub.ai/favicon.ico';
                header.appendChild(logo);

                // Create a new link element for the close icon
                const closeIcon = document.createElement('a');
                closeIcon.className = 'try-on-hub-ifrm-header-close-icon';
                closeIcon.innerHTML = '&#x2715;';
                header.appendChild(closeIcon);

                container.appendChild(header);

                // Create a new iframe element
                const iframe = document.createElement('iframe');
                iframe.setAttribute('src', `https://www.tryonhub.ai/try-on-plugin?src=${encodeURIComponent(images[0].src)}`);
                iframe.style.width = '485px';
                iframe.style.height = '700px';
                iframe.style.border = 'none';

                container.appendChild(iframe);

                document.body.appendChild(overlay);
                document.body.appendChild(container);

                overlay.addEventListener('click', function (e) {
                    this.parentNode.removeChild(container);
                    this.parentNode.removeChild(this);
                });
            }
        });
    }
})
```
