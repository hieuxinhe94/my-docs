# CDN (Content Delivery Network)

Để bắt đầu, bạn cần thêm plugin TryOnAI vào ứng dụng web của mình. Plugin TryOnAI hoạt động cả trong môi trường CommonJS
và AMD (Asynchronous Module Definition) và có thể truy cập thông qua không gian tên global của TryOnAI khi được tải qua
một CDN (Content Delivery Network).

Khi tích hợp Plugin TryOnAI qua CDN, nên sử dụng phiên bản được tối ưu hóa cho sản xuất (minified). Bạn có thể chỉ định phiên bản bằng cách thêm nó vào đường dẫn CDN như sau:

- [Đối với một phiên bản cụ thể](https://tryonai.com/tryonai-plugin-1.2.0.min.js)
- [Đối với phiên bản mới nhất (tự động cập nhật)](https://tryonai.com/tryonai-latest.min.js)

# Tải Plugin

Để tải Plugin thông qua CDN, hãy thêm đoạn mã sau vào giữa các thẻ ```<head>``` của ứng dụng web của bạn:

```html
<!-- Đối với một phiên bản cụ thể: -->
<script src='https://tryonai.com/tryonai-plugin-1.2.0.min.js'></script>

<!-- Đối với phiên bản mới nhất (tự động cập nhật): -->
<script src='https://tryonai.com/tryonai-latest.min.js'></script>
```

# Khởi Tạo TryOnAI
Sau khi tải plugin, khởi tạo TryOnAI trong mã JavaScript của bạn:
```js
// Khởi tạo TryOnAI với API key của bạn
var tryonaiInstance = TryOnAI.init({
  apiKey: 'YOUR_API_KEY',
  // Các tùy chọn khác...
});

// Gọi các hàm và sự kiện từ TryOnAI
tryonaiInstance.someFunction();
```
