# CDN (Content Delivery Network)

Để bắt đầu, bạn cần thêm plugin TryOnAI vào ứng dụng web của mình. Plugin TryOnAI hoạt động cả trong môi trường CommonJS
và AMD (Asynchronous Module Definition) và có thể truy cập thông qua không gian tên global của TryOnAI khi được tải qua
một CDN (Content Delivery Network).

Khi tích hợp Plugin TryOnAI qua CDN, nên sử dụng phiên bản được tối ưu hóa cho sản xuất (minified). Bạn có thể chỉ định phiên bản bằng cách thêm nó vào đường dẫn CDN như sau:

- [Đối với một phiên bản cụ thể](https://tryonai.com/tryonai-plugin-1.2.0.min.js)
- [Đối với phiên bản mới nhất (tự động cập nhật)](https://tryonai.com/tryonai-latest.min.js)

# Tải Plugin

Để tải Plugin thông qua CDN, hãy thêm đoạn mã sau vào giữa các thẻ ```<head>``` của ứng dụng web của bạn:

```html
<!-- Đối với một phiên bản cụ thể: -->
<script src='https://yourURL/tryonhub.min.js'></script>

<!-- Đối với phiên bản mới nhất (tự động cập nhật): -->
<script src='https://yourURL/tryonhub.min.js'></script>
```

# Khởi Tạo file JS config
tạo file tryonhub.min.js ở trong thư mục public
```js
// This code is wrapped in an immediately invoked function expression (IIFE) to create a separate scope
!(function () {
    // Get all elements with the class 'try-on-hub-image-src'
    const e = document.getElementsByClassName('try-on-hub-image-src');

    // Loop through each element
    for (var t = 0; t < e.length; t++) {
        // Create a new button element
        const n = document.createElement('button');

        // Add the class 'try-on-hub-btn' to the button
        n.classList.add('try-on-hub-btn');

        // Set the button text content to 'Try it on!'
        n.textContent = 'Try it on!';

        // Append the button to the current element
        e[t].appendChild(n);

        // Add a click event listener to the button
        n.addEventListener('click', function (e) {
            // Find the closest ancestor element with the class 'try-on-hub-image-src' and get all its 'img' children
            var t = e.target.closest('.try-on-hub-image-src').getElementsByTagName('img');

            // Check if there are any 'img' elements found
            if (t.length > 0) {
                // Create a new div element for the overlay
                var n = document.createElement('div');
                n.classList.add('try-on-hub-ifrm-overlay');

                // Create a new div element for the container
                var d = document.createElement('div');
                d.classList.add('try-on-hub-ifrm-container');

                // Create a new div element for the header
                var r = document.createElement('div');
                r.classList.add('try-on-hub-ifrm-header');

                // Create a new paragraph element for 'Powered by'
                var a = document.createElement('p');
                a.classList.add('try-on-hub-ifrm-header-power-by');
                r.appendChild(a);
                a.textContent += 'Powered by';

                // Create a new paragraph element for 'TryOnHub.AI'
                var o = document.createElement('p');
                o.classList.add('try-on-hub-ifrm-header-brand');
                r.appendChild(o);
                o.textContent += 'TryOnHub.AI';

                // Create a new image element for the logo
                var k = document.createElement('img');
                k.src = 'https://www.tryonhub.ai/favicon.ico';
                r.appendChild(k);

                // Create a new link element for the close icon
                var i = document.createElement('a');
                i.className = 'try-on-hub-ifrm-header-close-icon';
                i.innerHTML = '&#x2715;';
                r.appendChild(i);

                // Append the header to the container
                d.appendChild(r);

                // Create a new iframe element
                var c = document.createElement('iframe');

                // Set the 'src' attribute of the iframe to the TryOnHub.AI URL with the 'src' parameter as the encoded image source URL
                c.setAttribute('src', `https://www.tryonhub.ai/try-on-plugin?src=${encodeURIComponent(t[0].src)}`);

                // Set the width and height styles of the iframe
                c.style.width = '485px';
                c.style.height = '700px';

                // Set the border style of the iframe to 'none'
                c.style.border = 'none';

                // Append the iframe to the container
                d.appendChild(c);

                // Append the overlay and container to the body
                document.body.appendChild(n);
                document.body.appendChild(d);

                // Add a click event listener to the overlay to remove the container and overlay when clicked
                n.addEventListener('click', function (e) {
                    this.parentNode.removeChild(d);
                    this.parentNode.removeChild(this);
                });

```
